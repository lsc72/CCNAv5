Vérification du protocole EIGRP avec IPv4

La commande show ip protocols affiche les paramètres et d'autres informations sur l'état actuel des processus de routage IPv4 actifs configurés sur le routeur. La commande show ip protocols affiche les différents types de sorties spécifiques à chaque protocole de routage.

Le résultat présenté à la Figure 1 fournit plusieurs paramètres EIGRP :

1. Le protocole EIGRP est un protocole de routage dynamique actif sur R1 configuré avec le numéro de système autonome 1.

2. L'ID de routeur EIGRP de R1 est 1.1.1.1.

3. Les distances administratives EIGRP sur R1 sont de 90 en interne et de 170 en externe (valeurs par défaut).

4. Par défaut, le protocole EIGRP ne récapitule pas automatiquement les réseaux. Les sous-réseaux sont inclus dans les mises à jour de routage.

5. Les contiguïtés de voisinage EIGRP de R1 avec d'autres routeurs servent à recevoir les mises à jour de routage EIGRP.

Remarque : avant IOS 15, la récapitulation automatique EIGRP était activée par défaut.

Le résultat de la commande show ip protocols facilite le débogage des opérations de routage. Les informations présentées dans le champ Routing Information Sources (Sources d'informations de routage) permettent d'identifier un routeur soupçonné de fournir de mauvaises informations de routage. Le champ Routing Information Sources (Sources d'informations de routage) répertorie toutes les sources de routage EIGRP utilisées par le logiciel Cisco IOS pour créer sa table de routage IPv4. Pour chaque source, notez les valeurs suivantes :

  • Adresse IPv4

  • Distance administrative

  • Heure de réception de la dernière mise à jour depuis la source

Comme indiqué à la Figure 2, le protocole EIGRP dispose d'une distance administrative de 90 pour les routes internes et de 170 pour les routes importées à partir d'une source externe, notamment les routes par défaut. Comparé à d'autres protocoles IGP, le protocole EIGRP est le protocole privilégié par Cisco IOS, car il présente la distance administrative la plus courte. Le protocole EIGRP dispose d'une troisième valeur de distance administrative de 5, pour les routes récapitulatives.
