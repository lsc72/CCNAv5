Conventions d'attribution de noms

Avant la version du logiciel Cisco IOS 15.0, le modèle de package Cisco IOS comprenait huit packages pour les routeurs Cisco, comme indiqué dans la figure. Ce modèle de package a été lancé avec la catégorie principale du logiciel Cisco IOS 12.3, puis utilisé dans d'autres catégories. Le modèle de package d'image comprend huit images IOS, dont trois considérées comme des packages premium.

Les cinq packages non premium sont :

  • IP Base : IP Base est l'image du logiciel Cisco IOS d'entrée de gamme

  • IP Voice : Voix et données convergentes, voix sur IP, VoFR et téléphonie IP

  • Advanced Security : fonctionnalités de sécurité et de réseau privé virtuel, incluant pare-feu Cisco IOS, système IDS/système de protection contre les intrusions, IPsec, 3DES et réseau privé virtuel

  • SP (Service Provider) Services : ajoute SSH/SSL, ATM, VoATM et MPLS à IP Voice

  • Enterprise Base : protocoles d'entreprise : prise en charge d'Appletalk, IPX et IBM

Remarque : à partir de la famille de versions du logiciel Cisco IOS 12.4, SSH est disponible dans toutes les images.

Trois autres packages premium offrent d'autres combinaisons de fonctionnalités logicielles IOS qui répondent à des exigences réseau plus complexes. Toutes les fonctionnalités sont combinées dans le package Advanced Enterprise Services. Ce package intègre la prise en charge de tous les protocoles de routage avec les fonctions de voix, de sécurité et de réseau privé virtuel :

  • Advanced Enterprise Services : fonctionnalités complètes du logiciel Cisco IOS

  • Enterprise Services : Enterprise Base et Service Provider Services

  • Advanced IP Services : Advanced Security, Service Provider Services et prise en charge d'IPv6

Remarque : Cisco Feature Navigator est un outil qui permet de trouver le système d'exploitation Cisco approprié en fonction des fonctionnalités et technologies requises.
