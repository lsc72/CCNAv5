Contexte/scénario

Dans cet exercice, deux routeurs sont configurés de manière à communiquer l'un avec l'autre. Vous êtes chargé de configurer les sous-interfaces en vue de communiquer avec les commutateurs. Vous configurerez les VLAN, le trunking ainsi qu'EtherChannel avec PVST. Tous les périphériques Internet sont préconfigurés.
