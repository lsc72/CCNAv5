Schémas de topologie réseau

Les schémas de topologie réseau indiquent l'emplacement, la fonction et l'état des périphériques présents sur le réseau. Il existe deux types de schémas de topologie réseau, à savoir les schémas de topologie physique et les schémas de topologie logique.

Topologie physique

Un schéma physique du réseau indique la disposition physique des périphériques connectés au réseau. Il est nécessaire de savoir comment les périphériques sont physiquement connectés pour pouvoir résoudre les problèmes liés à la couche physique. Les informations figurant sur le schéma incluent généralement :

  • le type de périphérique 

  • le modèle et le fabricant 

  • la version du système d’exploitation 

  • le type et l’identificateur de câble 

  • les spécifications du câble 

  • le type de connecteur 

  • les points d'extrémité du câblage

Topologie logique

Une topologie réseau logique montre comment les périphériques sont connectés logiquement sur le réseau, indiquant ainsi comment ces périphériques transfèrent concrètement les données sur le réseau lorsqu'ils communiquent entre eux. Des symboles sont utilisés pour représenter les différents éléments du réseau, tels que les routeurs, les serveurs, les hôtes, les concentrateurs VPN et les périphériques de sécurité. De plus, les connexions entre divers sites peuvent être affichées, mais sans toutefois indiquer les emplacements physiques réels. Un schéma logique de réseau peut comprendre les informations suivantes :

  • Identificateurs de périphériques

  • Adresse IP et longueurs de préfixe

  • Identificateurs d’interfaces

  • Type de connexion

  • DLCI pour circuits virtuels

  • Réseaux privés virtuels de site à site

  • Protocoles de routage

  • Routes statiques

  • Protocoles de liaison de données

  • Technologies WAN utilisées
