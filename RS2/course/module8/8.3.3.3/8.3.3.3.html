Configuration du protocole OSPFv3 à zone unique

Vérification d'OSPFv3

Le moyen le plus rapide de vérifier les paramètres d'interface OSPF consiste à utiliser la commande show ipv6 opsf interface. Cette commande fournit une liste détaillée de chaque interface utilisant OSPF.

Pour extraire et afficher un résumé des interfaces compatibles OSPFv3 sur R1, utilisez la commande show ipv6 ospf interface brief, comme illustré dans la Figure 1.

Utilisez le contrôleur de syntaxe de la Figure 2 pour afficher un résumé des interfaces compatibles OSPF sur R2 en utilisant la commande show ipv6 ospf interface brief. Remarquez que la spécification du nom d'interface comme dans la commande show ipv6 ospf interface serial 0/0/1 permet d'obtenir des informations OSPF détaillées.

Continuez à utiliser le contrôleur de syntaxe de la Figure 2 pour obtenir un résumé des interfaces compatibles OSPF sur R3 en utilisant la commande show ipv6 ospf interface brief. Récupérez et affichez des informations supplémentaires pour l'interface série 0/0/0 au moyen de la commande show ipv6 ospf interface serial 0/0/0.
