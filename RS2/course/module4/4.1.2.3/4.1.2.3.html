Configuration initiale d'un routeur

Brancher les périphériques

Lorsque vous concevez un nouveau réseau ou que vous mappez un réseau existant, fournissez des explications écrites sur le réseau. Au minimum, la documentation doit identifier les éléments suivants :

  • les noms des périphériques ;

  • les interfaces utilisées ;

  • les adresses IP et masques de sous-réseau ;

  • les adresses des passerelles par défaut.

Comme l'illustre la figure, ces informations sont recueillies au moyen de deux documents réseau utiles :

  • Schéma de topologie : offre une référence visuelle présentant la connectivité physique et l'adressage de couche 3 logique. Il est souvent créé à l'aide d'un logiciel, tel que Microsoft Visio.

  • Table d'adressage : table dans laquelle figurent des noms de périphérique, interfaces, adresses IPv4, masques de sous-réseau et adresses de passerelle par défaut.
