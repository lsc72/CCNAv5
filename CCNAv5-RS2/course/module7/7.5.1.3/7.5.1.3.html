La table de routage

Parties d'une entrée de route IPv4

La figure illustre une entrée de table de routage IPv4 sur R1 pour la route vers le réseau distant 172.16.4.0 sur R3. L'entrée identifie les informations suivantes :

  • Origine de la route : indique comment la route a été apprise.

  • Réseau de destination : identifie l'adresse du réseau distant.

  • Distance administrative : indique la fiabilité de l'origine de la route.

  • Métrique : indique la valeur attribuée pour atteindre le réseau distant. Les valeurs inférieures indiquent les routes préférées.

  • Tronçon suivant : indique l'adresse IPv4 du prochain routeur auquel transférer le paquet.

  • Horodatage de route : détermine à quel moment la route a été détectée pour la dernière fois.

  • Interface de sortie : identifie l'interface de sortie à utiliser pour transférer un paquet vers la destination finale.
