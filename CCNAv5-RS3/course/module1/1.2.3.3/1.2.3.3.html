Gestion des périphériques

Une configuration de routeur de base inclut le nom d'hôte pour identification, des mots de passe pour la sécurité, l'affectation d'adresses IP à des interfaces pour la connectivité et un routage de base. La Figure 1 illustre les commandes saisies pour activer un routeur avec le protocole OSPF. Vérifiez et enregistrez les modifications de configuration, au moyen de la commande copy running-config startup-config. La Figure 2 indique les résultats des commandes de configuration qui ont été entrées dans la Figure 1. Pour effacer la configuration du routeur, utilisez la commande erase startup-config, puis la commande reload.

Dans la Figure 3, utilisez le contrôleur de syntaxe pour vérifier les configurations de routeurs au moyen de ces commandes show.
