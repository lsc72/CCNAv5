Gestion des périphériques

Quel que soit le périphérique réseau Cisco IOS implémenté, il existe deux méthodes pour connecter un PC à ce périphérique réseau en vue de tâches de configuration et de surveillance : la gestion hors bande et intrabande, comme illustré par la figure.

La gestion hors bande est utilisée pour la configuration initiale ou lorsqu'une connexion réseau n'est pas disponible. Pour effectuer une configuration grâce à la gestion hors bande, les éléments suivants sont requis :

  • connexion directe à la console ou au port auxiliaire

  • client d'émulation de terminal

La gestion intrabande est utilisée pour effectuer ou surveiller des changements de configuration sur un périphérique réseau, via une connexion réseau. Pour effectuer une configuration grâce à la gestion intrabande, les éléments suivants sont requis :

  • au moins une interface réseau sur le périphérique connectée et opérationnelle

  • le protocole Telnet, SSH ou HTTP pour accéder à un périphérique Cisco
