Sélection des périphériques réseau

Gestion des périphériques

Vous trouverez ci-dessous les commandes de la plate-forme logicielle IOS les plus couramment utilisées afin d'afficher et de vérifier l'état de fonctionnement du routeur et des fonctionnalités associées au réseau. Ces commandes sont regroupées en plusieurs catégories.

Associées au routage :

  • show ip protocols  : affiche des informations sur les protocoles de routage configurés. Si le protocole OSPF est configuré, elles comprennent l'ID de processus OSPF, l'ID du routeur, les réseaux annoncés par le routeur, les voisins dont le routeur reçoit les mises à jour et la distance administrative par défaut, à savoir 110 pour OSPF. (Figure 1)

  • show ip route  : affiche des informations sur les tables de routage, notamment les codes de routage, les réseaux connus, la distance administrative et les métriques, le moyen d'apprentissage des routes, le saut suivant, les routes statiques et les routes par défaut. (Figure 2)

  • show ip ospf neighbor  : affiche des informations sur les voisins OSPF appris, y compris l'ID de routeur du voisin, la priorité, l'état (Full signifiant que la contiguïté a été établie), l'adresse IP et l'interface locale apprise auprès du voisin. (Figure 3)

Associées à l'interface :

  • show interfaces  : affiche les interfaces avec l'état (de protocole) de ligne, la bande passante, le retard, la fiabilité, l'encapsulation, le mode bidirectionnel et les statistiques d'E/S. En cas de spécification sans la désignation d'une interface spécifique, toutes les interfaces seront affichées. Si une interface spécifique est désignée après la commande, seules les informations relatives à cette interface seront affichées. (Figure 4)

  • show ip interfaces  : affiche des informations sur les interfaces, notamment l'état du protocole, l'adresse IP, le fait qu'une adresse de diffusion (helper) soit configurée et si une liste de contrôle d'accès est activée pour l'interface. En cas de spécification sans la désignation d'une interface spécifique, toutes les interfaces seront affichées. Si une interface spécifique est désignée après la commande, seules les informations relatives à cette interface seront affichées. (Figure 5)

  • show ip interface brief  : affiche toutes les interfaces avec les informations d'adressage IP et l'état des interfaces et des protocoles de ligne. (Figure 6)

  • show protocols  : affiche des informations sur le protocole routé activé et l'état de protocole des interfaces. (Figure 7)

La commande show cdp neighbors (Figure 8) fait également partie des commandes relatives à la connectivité. Elle affiche des informations sur les périphériques connectés directement, notamment l'ID du périphérique, l'interface locale à laquelle le périphérique est connecté, la fonctionnalité (R = routeur, S = commutateur), la plate-forme et l'ID du port du périphérique distant. L'option de détails indique, entre autres, des informations sur l'adressage IP et la version d'IOS.
