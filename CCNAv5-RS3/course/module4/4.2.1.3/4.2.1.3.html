Fonctionnement LAN sans fil

Structure de trame 802.11

Remarque : les champs de type et de sous-type de trame sont utilisés pour indiquer le type de transmission sans fil concerné. Comme illustré à la figure, une trame sans fil peut être de l'un des trois types suivants :

  • Trame de gestion : utilisée dans la maintenance des communications, notamment la recherche, l'authentification et l'association à un point d'accès.

  • Trame de contrôle : utilisée pour faciliter l'échange de trames de données entre des clients sans fil.

  • Trame de données : utilisée pour transférer des informations sur les données utiles, par exemple les fichiers et les pages Web.
