Configuration LAN sans fil

Configurer un routeur sans fil

Le routeur sans fil Linksys EA6500 est fourni avec un CD de configuration.

Pour configurer et installer le logiciel du routeur Linksys EA6500, observez la procédure suivante :

Étape 1. Insérez le CD dans le lecteur CD ou DVD. La procédure d'installation doit démarrer automatiquement. Si le CD d'installation n'est pas fourni, téléchargez le programme d'installation sur le site http://Linksys.com/support.

La Figure 1 illustre l'écran de connexion initiale du routeur Linksys EA6500. Celui-ci contient des instructions pour raccorder l'alimentation et la connexion Internet du routeur.

Remarque : dans notre exemple, le routeur sans fil ne sera pas connecté à Internet.

Étape 2. Cliquez sur Next (Suivant) pour commencer l'installation.

Le programme d'installation lance le processus d'installation et affiche une fenêtre de statut (Figure 2). Pendant ce temps, le programme tente de configurer et d'activer la connexion Internet. Dans notre exemple, la connexion Internet n'est pas disponible et, après quelques tentatives, l'option permettant d'ignorer cette étape s'affiche.

L'écran de configuration des paramètres du routeur Linksys s'affiche (Figure 3). C'est ici que vous pourrez spécifier l'identifiant SSID, le mot de passe de connexion sans fil et le mot de passe d'administration.

Étape 3. Cliquez sur Next (Suivant) pour afficher l'écran récapitulatif des paramètres du routeur (Figure 4). Enregistrez ces paramètres si le tableau initial n'a pas été complété précédemment.

Étape 4. Cliquez sur Next (Suivant) pour afficher l'option de configuration de l'écran de compte Linksys Smart Wi-Fi (Figure 5).

Cet écran vous permet de gérer le routeur à distance depuis Internet. Dans cet exemple, le compte Linksys Smart Wi-Fi n'a pas été configuré, du fait de l'absence d'accès Internet.

Étape 5. Cliquez sur Continue (Continuer) pour afficher la fenêtre de connexion (Figure 6). Étant donné qu'aucune connexion Internet n'a été configurée, le mot de passe d'administration du routeur est demandé.

Étape 6. Une fois le mot de passe saisi, cliquez sur Log in (Se connecter) pour afficher la page d'accueil Linksys Smart Wi-Fi (Figure 7).
