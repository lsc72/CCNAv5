Fonctionnement du protocole EIGRP

Algorithme DUAL et table topologique

Comme indiqué à la Figure 1, la première ligne de la table topologique affiche :

  • P : la route est à l'état passif. Lorsque l'algorithme DUAL n'effectue pas ses calculs de diffusion pour déterminer le chemin d'un réseau, la route est en mode stable, également appelé état passif. Si DUAL recalcule ou recherche un nouveau chemin, la route est à l'état actif et affiche la lettre A. Toutes les routes de la table topologique d’un domaine de routage stable doivent être à l’état passif.

  • 192.168.1.0/24 : réseau de destination qui se trouve également dans la table de routage.

  • 1 successors : affiche le nombre de successeurs pour ce réseau. S'il existe plusieurs chemins à coût égal vers ce réseau, plusieurs successeurs sont disponibles.

  • FD is 3012096 : distance de faisabilité, la métrique EIGRP pour atteindre le réseau de destination. Il s'agit de la métrique affichée dans la table de routage IP.

Comme indiqué à la Figure 2, la première sous-entrée du résultat présente le successeur :

  • via 192.168.10.10 : adresse de tronçon suivant du successeur, R3. Cette adresse est indiquée dans la table de routage.

  • 3012096 : distance de faisabilité vers 192.168.1.0/24. Il s'agit de la métrique affichée dans la table de routage IP.

  • 2816 : distance annoncée du successeur et coût de R3 pour atteindre ce réseau.

  • Serial 0/0/1 : interface sortante servant à atteindre ce réseau, comme indiqué dans la table de routage.

Comme indiqué à la Figure 3, la deuxième sous-entrée indique le successeur potentiel, R1 (s'il n'y a pas de deuxième entrée, alors il n'y a pas d'autre successeur potentiel) :

  • via 172.16.3.1 : adresse de tronçon suivant du successeur potentiel, R1.

  • 41024256 : nouveau successeur potentiel de R2 vers 192.168.1.0/24, si R1 devient le nouveau successeur et devient la nouvelle métrique affichée dans la table de routage IP.

  • 2170112 : distance annoncée du successeur potentiel ou métrique de R1 pour atteindre ce réseau. La distance annoncée doit être inférieure à la distance de faisabilité actuelle de 3 012 096 pour remplir la condition de faisabilité.

  • Serial 0/0/0 : interface sortante servant à atteindre le successeur potentiel, si ce routeur devient le successeur.
