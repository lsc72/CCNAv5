Vérification du protocole EIGRP pour IPv6

La commande show ipv6 protocols affiche les paramètres et d'autres informations sur l'état des processus de routage IPv6 actifs actuellement configurés sur le routeur. La commande show ipv6 protocols affiche les différents types de sorties spécifiques à chaque protocole de routage IPv6.

Le résultat de la figure présente plusieurs paramètres EIGRP pour IPv6 précédemment décrits, notamment :

1. Le protocole EIGRP pour IPv6 est un protocole de routage dynamique actif sur R1 configuré avec le numéro de système autonome 2.

2. Voici les valeurs k utilisées pour calculer la métrique composite EIGRP. Les valeurs K1 et K3 sont réglées sur 1, par défaut, et les valeurs K2, K4 et K5 sur 0, par défaut.

3. L'ID de routeur EIGRP pour IPv6 de R1 est 1.0.0.0.

4. Comme pour le protocole EIGRP pour IPv4, les distances administratives du protocole EIGRP pour IPv6 présentent une valeur interne de 90 et une valeur externe de 170 (valeurs par défaut).

5. Les interfaces activées pour le protocole EIGRP pour IPv6.

Le résultat de la commande show ipv6 protocols facilite le débogage des opérations de routage. La section Interfaces indique les interfaces du protocole EIGRP pour IPv6 activées. Elle se révèle utile lorsque vous vérifiez que le protocole EIGRP est activé sur toutes les interfaces appropriées avec le numéro de système autonome correct.
